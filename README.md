### login-invite-reset

This is a PERN boilerplate for a web-app featuring a login and invite system. It features a session-based authentication system using Passport-local and also includes an admin panel where authorized users can invite new users and reset their password using the Sendgrid API.

## Stack used:

- React
- Node / Express
- Express-sessions with Redis as session store
- Passport-local for user authentication
- Bcrypt for hashing password
- Postgres

### PERN login-invite-reset

This is a boilerplate application featuring a login and invite system. 

It features a session-based authentication system using Passport-local and also includes an admin panel where authorized users can invite new users and send password reset requests using the Sendgrid API.

## Stack used:

- React
- Node / Express
- Express-session with Redis as session store
- Passport-local for user authentication
- Bcrypt for hashing password
- Postgres
